The Roadmap to AI Regulation

[x]
About

AI is beginning to generate new categories of risk—deepfakes, synthetic fraud, data flows regulators are only starting to name. Identity is shifting into something generated in real time, almost a new sense for a dimension we're only beginning to perceive. For The Roadmap to AI Regulation, part of Sumsub Multiverse, we built an identity around that idea—a glitch-kaleidoscope object where shifting color half-reveals a reality still forming behind it.

Challenge

As a standalone project within Sumsub Multiverse, this identity didn't need its own scalable system—that role already belongs to the wider brand. It needed one confident object that felt like AI-driven identity, not just represented it. A pure glitch reads as chaos; behind the distortion had to be something active and changing, not glitch for its own sake. The task was to create one abstract, generative object that feels alive and slightly dangerous, anchored to the brand's dark, futuristic palette.

Solution

We built the identity around a single object—a glitch-kaleidoscope: a fractured, barcode-like form concealing more than it shows, with something shifting behind its layers. Data is carried through color—transitioning tones read like signals moving between states, set against a dark, futuristic base with bright brand highlights and a couple of added accents. Layering creates a sense of motion—flaring in places like neural activity, reading elsewhere as zones instinctively perceived as unstable. As a standalone project within Multiverse, the identity didn't need to scale beyond this object—just to be bold and self-contained.
