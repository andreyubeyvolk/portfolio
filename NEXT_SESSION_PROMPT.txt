Привет. Продолжаем проект страницы портфолио в VS Code.

Проект лежит здесь:
C:\Projects\my-new-app

Перед началом прочитай файл:
PROJECT_NOTES.md

Основные файлы:
index.html
styles.css
assets/

Это статичная HTML/CSS-страница, сервер не нужен. Открывать можно напрямую index.html в браузере.

Важно сохранить текущие принципы:
- 24-колоночная сетка;
- margin и gutter по 16px;
- Inter, 16px текст, line-height 20px;
- 1-4 колонки: лого и меню;
- 5-8 колонки: описание проекта;
- 9-24 колонки: content frame;
- скролл справа внутри 16px зоны: 4px + 8px черный скролл + 4px, без стрелок и скруглений;
- hover underline у меню и ссылок слева направо;
- [x] выровнен по правому краю фото, hover: черный фон, белый текст.

Сначала проверь текущие файлы и не ломай уже настроенные pixel-level правки.

Как открыть в VS Code:

cd C:\Projects\my-new-app
code .

Как посмотреть страницу:

Открой файл index.html в браузере.
Если в VS Code стоит расширение Live Server, можно нажать правой кнопкой по index.html и выбрать Open with Live Server, но для этой страницы это необязательно.
